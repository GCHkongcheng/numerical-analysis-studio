## 三、核心问题与优化建议

### 3.1 CSS 架构混乱 ⭐⭐⭐

**问题**:
1. **样式分散**: 全局 CSS 类 + Tailwind utilities + 内联样式混用
2. **命名不一致**: `studio-card`, `nav-tab`, `operation-pill` 等命名缺乏统一规范
3. **可维护性差**: 234 个 className 散落在 3,402 行代码中

**建议**:

#### 方案 A: 采用 CSS Modules（推荐）

将组件样式模块化，避免全局污染：

```typescript
// src/components/matrix/MatrixGrid.module.css
.grid {
  display: grid;
  gap: 0.5rem;
}

.cell {
  border: 1px solid rgb(15 23 42 / 0.12);
  border-radius: 0.76rem;
  background: white;
}

.cellPivot {
  background: rgb(255 237 213);
  border-color: rgb(251 146 60 / 0.8);
}

// MatrixGrid.tsx
import styles from './MatrixGrid.module.css';

<div className={styles.grid}>
  <div className={cn(styles.cell, isPivot && styles.cellPivot)} />
</div>
```

**优势**:
- 样式作用域隔离
- 类名自动哈希，避免冲突
- 更好的代码组织

#### 方案 B: 使用 Tailwind 的 @apply（次选）

将重复的 utility 组合提取为语义化类：

```css
/* globals.css */
.card {
  @apply rounded-2xl border border-slate-900/10 bg-white/88 p-5 shadow-lg;
}

.btn-primary {
  @apply rounded-full bg-gradient-to-r from-orange-600 to-orange-500 
         px-4 py-2 text-sm font-semibold text-white 
         transition-all hover:brightness-105;
}
```

**注意**: Tailwind 4.0 可能对 @apply 支持有变化，需要验证兼容性。

#### 方案 C: 建立设计令牌系统（长期）

使用 CSS 变量 + Tailwind 配置统一管理设计令牌：

```css
/* globals.css */
@theme inline {
  /* 间距系统 */
  --spacing-xs: 0.25rem;
  --spacing-sm: 0.5rem;
  --spacing-md: 1rem;
  --spacing-lg: 1.5rem;
  --spacing-xl: 2rem;
  
  /* 圆角系统 */
  --radius-sm: 0.5rem;
  --radius-md: 0.75rem;
  --radius-lg: 1rem;
  --radius-xl: 1.5rem;
  --radius-full: 9999px;
  
  /* 阴影系统 */
  --shadow-sm: 0 1px 2px rgb(15 23 42 / 0.08);
  --shadow-md: 0 4px 6px rgb(15 23 42 / 0.1);
  --shadow-lg: 0 10px 25px rgb(15 23 42 / 0.15);
}
```

**优先级**: 高  
**预期收益**: 显著提升样式可维护性，减少重复代码

### 3.3 性能优化空间 ⭐⭐

**问题**:
1. 大量 box-shadow 和 backdrop-filter 可能影响性能
2. 渐变背景在低端设备上可能卡顿
3. 矩阵网格在大尺寸时渲染压力大

**建议**:

#### 1. 使用 CSS containment

```css
.studio-card {
  contain: layout style paint;  /* 隔离布局和绘制 */
}

.matrix-grid {
  contain: layout;  /* 隔离布局计算 */
}

.matrix-cell {
  contain: paint;  /* 隔离绘制 */
}
```

#### 2. 优化阴影和模糊效果

```css
/* 使用 will-change 提示浏览器优化 */
.studio-card {
  will-change: transform;  /* 仅在需要动画时使用 */
}

/* 简化阴影层级 */
.studio-card {
  /* 原来: 多层阴影 */
  /* box-shadow: 0 10px 25px -22px rgb(15 23 42 / 0.65),
                inset 0 0 0 1px rgb(255 255 255 / 0.65); */
  
  /* 优化: 单层阴影 */
  box-shadow: 0 4px 12px rgb(15 23 42 / 0.15);
}
```

#### 3. 虚拟滚动大型矩阵

对于超大矩阵（如 100x100），使用虚拟滚动：

```typescript
// 使用 react-window 或 @tanstack/react-virtual
import { useVirtualizer } from '@tanstack/react-virtual';

function VirtualMatrixGrid({ matrix }: { matrix: string[][] }) {
  const parentRef = useRef<HTMLDivElement>(null);
  
  const rowVirtualizer = useVirtualizer({
    count: matrix.length,
    getScrollElement: () => parentRef.current,
    estimateSize: () => 40,
  });
  
  return (
    <div ref={parentRef} className="matrix-scroll" style={{ height: '400px' }}>
      <div style={{ height: `${rowVirtualizer.getTotalSize()}px` }}>
        {rowVirtualizer.getVirtualItems().map((virtualRow) => (
          <div key={virtualRow.index} className="matrix-row">
            {/* 渲染行内容 */}
          </div>
        ))}
      </div>
    </div>
  );
}
```

**优先级**: 中  
**预期收益**: 提升低端设备性能，支持更大规模矩阵
