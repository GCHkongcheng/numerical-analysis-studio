### 2.3 可访问性增强

#### A. 全局焦点样式 (globals.css)

```css
/* 所有可聚焦元素 */
*:focus-visible {
  outline: 2px solid var(--accent);
  outline-offset: 2px;
}

/* 主按钮焦点 */
.studio-primary-btn:focus-visible {
  outline: 2px solid white;
  outline-offset: 2px;
  box-shadow: 0 0 0 4px rgb(234 88 12 / 0.3);
}

/* 按钮点击反馈 */
.studio-primary-btn:active {
  transform: scale(0.98);
}

/* 导航标签、操作按钮、模式切换焦点 */
.nav-tab:focus-visible,
.operation-pill:focus-visible,
.mode-chip:focus-visible,
.step-control:focus-visible {
  outline: 2px solid var(--accent);
  outline-offset: 2px;
}

/* 输入框焦点 */
.studio-input:focus-visible,
.studio-select:focus-visible {
  outline: 2px solid var(--accent);
  outline-offset: 1px;
  border-color: rgb(234 88 12 / 0.6);
}

/* 矩阵输入焦点保持原有样式 */
.matrix-input:focus-visible {
  outline: none;
  border-color: rgb(234 88 12 / 0.6);
  box-shadow: 0 0 0 3px rgb(251 146 60 / 0.25);
}
```

**特性**:
- 使用 `:focus-visible` 伪类（仅键盘导航时显示）
- 统一的橙色焦点指示器
- 不同元素类型有适配的焦点样式
- 符合 WCAG 2.1 焦点可见性要求

#### B. ARIA 属性增强 (MatrixGrid.tsx)

**为矩阵网格添加完整的 ARIA 支持**:

```typescript
// 网格容器
<div
  role="grid"
  aria-label={editable ? "可编辑矩阵输入网格" : "矩阵显示网格"}
  aria-rowcount={rowCount}
  aria-colcount={colCount}
  aria-readonly={!editable}
>

// 每个单元格
<input
  role="gridcell"
  aria-rowindex={r + 1}
  aria-colindex={c + 1}
  aria-label="第 X 行, 第 Y 列, 主元/高亮行/增广列"
/>
```

**ARIA 属性说明**:
- `role="grid"`: 标识为网格结构
- `aria-label`: 提供网格描述
- `aria-rowcount/colcount`: 声明行列数
- `aria-readonly`: 标识是否可编辑
- `role="gridcell"`: 标识单元格
- `aria-rowindex/colindex`: 单元格位置
- 动态 `aria-label`: 包含位置和状态信息

**屏幕阅读器体验**:
- 可以正确识别矩阵结构
- 能够朗读单元格位置
- 能够识别特殊状态（主元、高亮、增广列）

### 2.4 颜色对比度优化 (globals.css)

**修改内容**:

```css
/* 原来 */
color: rgb(71 85 105);  /* #475569 - 对比度可能不足 */

/* 优化后 */
color: var(--color-neutral-700);  /* #334155 - 更高对比度 */
```

**应用位置**:
- `.nav-tab` - 导航标签文字
- `.operation-pill` - 操作按钮文字
- `.mode-chip` - 模式切换按钮文字

**对比度检查**:
- 原来: `#475569` on `#ffffff` = 约 4.2:1 (接近 WCAG AA 边界)
- 优化后: `#334155` on `#ffffff` = 约 8.5:1 (超过 WCAG AAA 标准)

### 2.5 性能优化

#### 移动端动画优化
```css
@media (max-width: 640px) {
  @media (prefers-reduced-motion: no-preference) {
    * {
      animation-duration: 0.01ms !important;
      transition-duration: 0.01ms !important;
    }
  }
}
```

#### 尊重用户偏好
```css
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }
}
```

**优势**:
- 提升低端设备性能
- 尊重用户的无障碍偏好
- 减少电池消耗
